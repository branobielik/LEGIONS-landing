🎴 How to Play

LEGIONS TCG — choose your battle

Typhon → Outpost Frontline Last Stand

⚔️ Skirmish

🃏 Deck

Each player uses a deck of 7 cards:

⭐Legionvalue 3×1
💎Elitevalue 2×2
🔷Recruitvalue 1×4
⚔️ Match

First to win 2 sets wins the match (Best of 3).

🃏 Set — 7 duels
1
Each player secretly orders their 7 cards in a row.
2
Both players reveal simultaneously.
3
Cards are compared position by position (left to right).
🥊 Duel
  • Higher card value wins the duel → 1 point.
  • 🤝 Tie = 0 points for both.
🏆 Set Result
  • More points → wins the set.
  • 🤝 Set tie goes to Earthlings.
⏱️ Overtime
Both players draw 1 card at random and reveal simultaneously.
Higher value wins the set. If equal → draw again.
If all cards are used and still tied → Earthlings win the set.
🌍 Sides

Sides alternate between sets:

SET 1
Player A = 🌍 Earthlings
Player B = 🤖 Synthbots
SET 2
Player A = 🤖 Synthbots
Player B = 🌍 Earthlings
SET 3
same as Set 1

🏰 Siege

🃏 Deck

Each player uses a deck of 7 cards:

⭐Legionvalue 3×1
💎Elitevalue 2×2
🔷Recruitvalue 1×4
⚔️ Match

First to win 2 sets wins the match (Best of 3).

🧱 Siege Pressure
  • If a player wins 2 duels in a row, they gain +1 bonus point.
  • This bonus can trigger multiple times during a set.
  • 🤝 A tied duel breaks the streak for both players.
🃏 Set — 7 duels
1
Each player secretly orders their 7 cards.
2
Both players reveal simultaneously.
3
Cards are compared position by position (left to right).
🥊 Duel
  • Higher card value wins the duel → 1 point.
  • 🤝 Tie = 0 points for both.
🏆 Set Result
  • Player with more points (including Siege bonus points) wins the set.
  • If still tied after all duels, use Overtime.
⏱️ Overtime
Both players draw 1 card at random and reveal simultaneously.
Higher value wins the set. If equal → draw again.
If all cards are used and still tied → Earthlings win the set.
🌍 Sides

Sides alternate between sets, same as Skirmish.
